About HYDRAFloods

HYDRAFloods is a publicly available web-based near real-time flood monitoring tool. It is designed to improve flood monitoring system using freely available satellite remote sensing data. This tool is benefitted by applying Artificial intelligence (AI) technique to detect floods more accurately. This tool helps emergency response and relief sectors by providing more accurate flood information and it's extent.
